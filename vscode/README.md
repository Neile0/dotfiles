# Kelvin Systems — VSCode Theme

## Option 1 — VSIX install (recommended)

**Via command line:**
```bash
code --install-extension kelvin-systems.vsix
```

**Via UI:**
`Extensions` sidebar → `···` menu → `Install from VSIX` → select `kelvin-systems.vsix`

Then: `Cmd+Shift+P` → **Color Theme** → **Kelvin Systems**

## Option 2 — Manual install

```bash
cp -r . ~/.vscode/extensions/kelvin-systems-theme
```

Restart VSCode, then: `Cmd+Shift+P` → **Color Theme** → **Kelvin Systems**
